FEATURES

Transfer files between all devices on the same wireless Wi-Fi network.

  • Mobile Gallery

    It can send photos and videos from the gallery of your Android and Ios devices. You can save your media files from other devices to the gallery.

  • Multiple & Large Files

    You can send and receive multiple files at the same time. You can easily send large files to other devices.

  • File Formats

    It supports all file formats. Pdf, Doc, Mp3, Zip, Apk etc. You can save all files on your mobile device and use them from the file manager.

  • Connecting a Device

    Automatically finds your device on your network. Other devices are listed automatically when you open the application.

  • File Sharing

    Files downloaded to mobile devices can be sent by email and shared with another application.

  • Safe & Fast

    It does not require internet connection during file transfer. You can transfer files from your local Wi-Fi network regardless of internet speed. Send the file you want and wait for the other device to accept the request. Secure file transfer between devices.
